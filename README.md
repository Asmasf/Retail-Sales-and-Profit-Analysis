# 📊 Retail Sales & Profit Analysis

## Overview
This project analyzes retail sales transactions to identify profit drivers and support business decisions through exploratory data analysis and interactive Power BI dashboards.

## Tools
- Python
- Power BI

## Key Insights
- Full Price transactions generate the highest profits.
- December records the strongest sales performance.
- A few product categories contribute most of the company's profit.
- Sales trends reveal opportunities for profit optimization.

## Recommendations
- Prioritize high-profit product categories.
- Prepare inventory before peak sales periods.
- Improve underperforming categories.
- Align sales strategies with profitability goals.
